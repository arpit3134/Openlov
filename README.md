# Openlov

A modern, production-ready AI Tools Directory & Blog Website built with Next.js 14, TypeScript, and Tailwind CSS.

## Features

- 🚀 Next.js 14 with App Router
- 📱 Fully Responsive Design
- 🎨 Tailwind CSS Styling
- 📝 Blog System with Dynamic Routes
- 🛠️ AI Tools Directory with Categories
- 🔍 SEO Optimized
- ⚡ Performance Optimized
- 🎯 Modern UI Components
- 📊 Search Functionality
- 📋 Trending Tools

- Node.js 18+
